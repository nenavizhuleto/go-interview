Правила набора

RRR ➜ Вызов абонента, где:
RRR — код квартиры абонента данного дома.
000 ➜ Вызов консьержа данного подъезда.

RRR ➜ Вызов абонента, где:
RRR — код квартиры абонента данного дома.
SSS BBBB RRR ➜ Вызов произвольного абонента, где:
SSS — код улицы абонента,
BBBB — код дома абонента,
RRR — код квартиры абонента.
SSS BBBB RRR T N ➜ Звонок на произвольное устройство, где:
SSS — код улицы,
BBBB — код дома,
RRR — код квартиры,
T — код типа устройства,
N — порядковый номер устройства.

Домофон калитки (многоабонентская панель)
BBBB RRR ➜ Вызов абонента, где:
BBBB — код дома,
RRR — код квартиры абонента.
BBBB PP ➜ Вызов консьержа, где:
BBBB — код дома,
PP — код подъезда консьержа.

RRR ➜ Вызов абонента, где:
RRR — код квартиры абонента того же дома.
000 ➜ Вызов консьержа подъезда абонента.
SSS BBBB RRR ➜ Вызов произвольного абонента, где:
SSS — код улицы абонента,
BBBB — код дома абонента,
RRR — код квартиры абонента.

SSS BBBB RRR ➜ Вызов произвольного абонента, где:
SSS — код улицы абонента,
BBBB — код дома абонента,
RRR — код квартиры абонента.
SSS BBBB RRR T N ➜ Звонок на произвольное устройство, где:
SSS — код улицы,
BBBB — код дома,
RRR — код квартиры,
T — код типа устройства,
N — порядковый номер устройства.
